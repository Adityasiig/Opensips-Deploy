# examples/

Sanitized templates of the files expected in `bundle/`. Use as a starting
point:

1. Copy `opensips.cfg.example` → `bundle/opensips.cfg`, then replace every
   `CHANGE_ME_*` placeholder with your real value.
2. Copy the other `*.example` files if you need them.
3. Generate your own `opensips-cp.tar.gz` and `opensips_dump.sql` from a
   working OpenSIPS server (see the root `README.md` for the commands).
